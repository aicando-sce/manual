Claude Project에서 문항 생성

학습 프롬프트 → 분석 결과 확인 → 문항 생성 프롬프트 입력
  • 1 Claude Project 접속 후 대화창에 학습 프롬프트 붙여넣기
    PDF와 함께 첨부해서 전송
    조판 완료한 개념정리 pdf를 사용하는 경우,
    개념유닛과 개념유닛코드를 함께 전송
  • 2 AI의 개념 분석 결과 확인
    개념코드, 핵심 포인트, 예시 이미지 목록 등
  • 3 같은 대화창에 문항 생성 프롬프트 붙여넣기
같은 대화창을 유지하는 것이 중요합니다.
학습 컨텍스트가 유지된 상태에서 문항이 생성됩니다.
